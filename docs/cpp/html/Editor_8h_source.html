28 /*
29  * Here's the basic logic that we use in SemGen to determine which OPB physical property is represented by a <species> in an SBML model:
30  *
31  * Using libSBML, we first look up the substance base units for the model and also whether the species's "hasOnlySubstanceUnits" attribute is true...
32  *
33  * if the substance base unit is mole
34  * if hasOnlySubstanceUnits for species is TRUE, then use OPB:Chemical_molar_amount (OPB_00425)
35  * else use OPB:Chemical_concentration (OPB_00340)
36  *
37  * else if the substance base unit is item
38  * if hasOnlySubstanceUnits for species is TRUE, then use OPB:Particle_count (OPB_01001)
39  * else use OPB:Particle_concentration (OPB_01000)
40  *
41  * else if the base unit is kilogram or gram
42  * if hasOnlySubstanceUnits is TRUE, then use OPB:Mass_of_solid_entity (OPB_01226)
43  * else
44  * if the spatial dimensions of the compartment containing the species = 1, use OPB:Mass_lineal_density (OPB_00190)
45  * else, if the spatial dimensions of the compartment containing the species = 2, use OPB:Mass_areal_density (OPB_00258)
46  * else, if the spatial dimensions of the compartment containing the species = 3, use OPB:Mass_volumetric_density (OPB_00101)
47  *
48  * Hope that's helpful for determining which OPB property to use in composite annotations for SBML <species>.
49  *
50  * Any questions, just lemme know.
51  */
